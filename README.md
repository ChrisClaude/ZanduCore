# Zandu Core 
This is the core api of Zandu CMS. 

## Getting Started
### Installation
Please make sure to have Git and .NET 5 installed.
```bash
git clone https://github.com/ChrisClaude/ZanduCore
```

```bash
cd ZanduCore
```

```bash
dotnet build
```

```bash
dotnet run
```
